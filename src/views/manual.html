Galacta Terminal Simulator Manual

Welcome to the Galacta Terminal Simulator for managing your spaceship fleet! Below you will find a list of commands you can use to interact with the system.

Available Commands

Ship Management

createShip [fleetID] [arguments]
    - Creates a new ship in the specified fleet.
    - Arguments: name, capacity, maxSpeed, type, weaponSystems, frontalShield, lateralLeftShield, lateralRightShield, backShield, shieldType.

listShips -a
    - Lists all ships with their names and IDs.

listShips -f [fleetID]
    - Lists ships associated with the specified fleet.

fetch [id]
    - Fetches details of a ship or fleet by its ID.

fetch -m [name]
    - Fetches details of a ship model by its name.

updateShip [fleetID] [shipID]
    - Initiates a command line interface to update ship details.

deleteShip [shipID]
    - Deletes the specified ship from the system.

Fleet Management

listFleets
    - Lists all fleets with their names, IDs, and generals.

updateFleet [fleetID]
    - Initiates a command line interface to update fleet details.

deleteFleet [FleetID]
    - Deletes the specified fleet from the system.

Ship Model Management

createShipModel [arguments]
    - Creates a new ship model based on the provided arguments.

listShipModels
    - Lists all saved ship models.

updateShipModel [arguments]
    - Initiates a command line interface to update ship model details.

deleteModel [ModelID]
    - Deletes the specified ship model from the system.

Usage Examples

Here are a few examples of how to use the commands:

> createShip fleet123 "Battleship" 500 30 "warship" "laser" 30 30 20 20 "energy"
> listShips -a
> fetch 123456
> deleteShip 123456

Notes

  • Ensure that all required arguments are provided for each command.
  • IDs are unique identifiers for ships and fleets; ensure no duplicates exist.
  • Use the fetch command to view details before updating or deleting.
